# apps/web

Blazor Web App frontend for MediaCloud, using MudBlazor.

## Current scaffold
- MudBlazor package installed and registered in DI
- Mud providers/layout wired in `MainLayout.razor`
- MudBlazor static assets linked in `App.razor`

## Responsibilities
- Authentication UI
- Dashboard and issue lists
- Issue landing pages
- Media detail pages
- Request submission/status UI
- RBAC-aware route/action guards
